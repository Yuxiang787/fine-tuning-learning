# Apple Silicon (M1/M2/M3/M4) 配置说明

## 设备检测

项目已自动适配 Apple Silicon 芯片，使用 MPS (Metal Performance Shaders) 后端加速。

```python
# 自动检测
if torch.backends.mps.is_available():
    device = torch.device("mps")  # Apple Silicon
elif torch.cuda.is_available():
    device = torch.device("cuda")  # NVIDIA GPU
else:
    device = torch.device("cpu")
```

## MacBook Air M4 24GB 推荐配置

### 推荐环境版本（2026-04）

- Python `3.11`（最低 `3.10`）
- torch `>=2.10.0`
- transformers `>=5.3.0`
- peft `>=0.18.1`
- datasets `>=4.7.0`
- accelerate `>=1.13.0`

### 第一阶段：极简入门

```python
# 01-极简入门/lora_finetune.py

LORA_ALPHA = 32
```


## 可选模型对比

| 模型 | 大小 | 中文支持 | 训练时间 (M4) | 推荐场景 |
|------|------|----------|---------------|----------|
| facebook/opt-125m | 125M | 弱 | ~5 分钟 | 流程测试 |
| Qwen/Qwen2.5-0.5B | 0.5B | 强 | ~15 分钟 | 中文任务（推荐） |
| TinyLlama-1.1B | 1.1B | 中 | ~30 分钟 | 进阶使用 |

### 训练与推理建议（M4 24GB）

1. 默认训练：`Qwen/Qwen2.5-0.5B + LoRA(r=16, alpha=32, batch_size=8, max_length=512)`
2. 全量微调：`Qwen/Qwen2.5-0.5B + batch_size=1 + gradient_accumulation_steps=8`
3. 推理：`temperature=0.7, top_p=0.9, max_new_tokens=256`，MPS 下开启 `torch.autocast(device_type="mps", dtype=torch.float16)`

## 调优建议

如果遇到显存不足：
1. 减小 `batch_size`: 8 → 4 → 2
2. 减小 `max_length`: 512 → 256
3. 使用更小的模型

如果资源充足想提升效果：
1. 增加 `lora_r`: 16 → 32
2. 增加 `batch_size`: 8 → 16
3. 增加训练轮数：3 → 5
