# 数据准备指南

## 数据格式

### 标准格式（推荐）

```json
{"instruction": "任务描述", "input": "输入内容", "output": "期望输出"}
```

**示例**：
```json
{"instruction": "将以下中文翻译成英文", "input": "你好，世界！", "output": "Hello, world!"}
{"instruction": "判断句子情感", "input": "这部电影太精彩了", "output": "正面"}
{"instruction": "回答数学问题", "input": "15 + 27 = ?", "output": "42"}
```

### 无输入格式

当任务不需要额外输入时，`input` 可以为空或省略：

```json
{"instruction": "今天星期几？", "input": "", "output": "今天星期五"}
{"instruction": "你是谁？", "output": "我是一个 AI 助手"}
```

```

转换后:
{"instruction": "解释以下概念", "input": "机器学习", "output": "机器学习是..."}
```

## 数据质量要求

### 1. 格式一致

```json
// ✓ 好：所有字段都有
{"instruction": "...", "input": "...", "output": "..."}

// ✗ 差：字段缺失或不一致
{"instruction": "...", "output": "..."}  // 缺少 input
{"instr": "...", "input": "..."}  // 字段名不对
```

### 2. 指令清晰

```json
// ✓ 好：任务明确
{"instruction": "将以下中文翻译成英文", ...}
{"instruction": "判断以下句子的情感倾向", ...}

// ✗ 差：任务模糊
{"instruction": "处理这个", ...}
{"instruction": "做一下", ...}
```

### 3. 输出准确

```json
// ✓ 好：答案准确完整
{"instruction": "1+1=?", "output": "2"}

// ✗ 差：答案错误或不完整
{"instruction": "1+1=?", "output": "大概 3 吧"}
```

### 4. 长度适中

| 任务类型 | 建议长度 |
|----------|----------|
| 翻译 | 10-100 字 |
| 问答 | 50-200 字 |
| 摘要 | 原文 1/3-1/2 |
| 创作 | 100-500 字 |

### 2. 数据清洗

```python
# clean_data.py
import json

def clean_data(input_path, output_path):
    with open(input_path, 'r', encoding='utf-8') as f_in, \
         open(output_path, 'w', encoding='utf-8') as f_out:
        for line in f_in:
            data = json.loads(line)

            # 去除空白
            data['instruction'] = data['instruction'].strip()
            data['output'] = data['output'].strip()
            if 'input' in data:
                data['input'] = data['input'].strip()

            # 过滤空数据
            if not data['instruction'] or not data['output']:
                continue

            f_out.write(json.dumps(data, ensure_ascii=False) + '\n')
```

### 3. 数据划分

```python
# split_data.py
from sklearn.model_selection import train_test_split
import json

def split_data(input_path, train_path, val_path, test_path):
    # 加载数据
    data = [json.loads(line) for line in open(input_path)]

    # 划分
    train, temp = train_test_split(data, test_size=0.3, random_state=42)
    val, test = train_test_split(temp, test_size=0.5, random_state=42)

    # 保存
    for path, split in [(train_path, train), (val_path, val), (test_path, test)]:
        with open(path, 'w', encoding='utf-8') as f:
            for item in split:
                f.write(json.dumps(item, ensure_ascii=False) + '\n')
```

### 4. 数据统计

```python
# 统计信息
def analyze_data(path):
    data = [json.loads(line) for line in open(path)]

    print(f"样本数：{len(data)}")
    print(f"指令平均长度：{sum(len(d['instruction']) for d in data)/len(data):.1f}")
    print(f"输入平均长度：{sum(len(d.get('input','')) for d in data)/len(data):.1f}")
    print(f"输出平均长度：{sum(len(d['output']) for d in data)/len(data):.1f}")
```

## 数据增强

### 1. 指令重写

```
原指令：将以下中文翻译成英文
变体 1：请把这句话译成英语
变体 2：中文翻英文
变体 3：帮我翻译
```

### 2. 数据回译

```
原文 → 翻译 → 回译 → 对比
你好世界 → Hello World → 你好世界 ✓
```

### 3. 提示词多样化

```python
templates = [

    "<user>\n{instruction}\n{input}\n</user>\n<assistant>\n{output}</assistant>",
    "[INST] {instruction} {input} [/INST] {output}",
]
```

## 常见问题

### Q1: 数据量太少怎么办？

**A**:
- 至少准备 50-100 条高质量样本
- 使用数据增强技术
- 考虑 Few-shot Learning

### Q2: 数据格式不统一怎么办？

**A**:
- 编写转换脚本统一格式
- 手动检查修正异常数据
- 建立数据录入规范

### Q3: 需要多少验证数据？

**A**:
- 训练集的 10-20%
- 最少 20-50 条
- 确保覆盖所有任务类型

### Q4: 中文数据需要特殊处理吗？

**A**:
- 选择支持中文的模型（如 Qwen、ChatGLM）
- 确保 tokenizer 支持中文
- 注意编码使用 UTF-8
